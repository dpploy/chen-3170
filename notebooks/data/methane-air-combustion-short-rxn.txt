#

# to form a second peroxy radical, which can react with #another methane molecule to continue the 
# propagation cycle.
#    
CH3* + O2 -> CH3O2*

CH3O2* + O2 -> CH3O2 + 2 O*
#
# Termination: The reaction terminates when two radicals react with each other to form a stable molecule.
